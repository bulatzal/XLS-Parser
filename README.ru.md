# XLS Parser
[[English]](README.md) [[Русский]](README.ru.md)

Парсер для работы с данными в формате .xls / .xlsx.

## Требования
Python 3.10 или новее и библиотека openpyxl. Установка:
```
pip install openpyxl
```
